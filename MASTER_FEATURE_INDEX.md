## üêõ Common Issues & Solutions

### Issue 1: No detection happening
**Symptom:** Messages sent but nothing extracted

**Solution:**
```bash
# Check API key
grep OPENAI_API_KEY .env.local

# Should output: OPENAI_API_KEY=sk-...
```

### Issue 2: URL scraping fails
**Symptom:** "Failed to scrape job URL" error

**Solution:**
- Verify URL is public and accessible
- Check internet connection
- Try a different URL from same job board
- Review browser console for errors

### Issue 3: Build errors
**Symptom:** TypeScript or build errors

**Solution:**
```bash
# Clean and rebuild
rm -rf .next
npm run build

# Check for TypeScript errors
npx tsc --noEmit
```
